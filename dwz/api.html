API使用文档
API地址：
http://shortenapi.sinaapp.com/api.php (POST) http://shortenapi.sinaapp.com/api_get.php?longurl= (GET)

数据返回类型：
JSON

请求限制：
每日接受10W次成功任务,待此API找到好归宿自然扩容!

返回数据示例：
{"Code":"0","Baidu":null,"Bailu":"http:\/\/bai.lu\/y0fiG","Bitly":"http:\/\/bit.ly\/xB3Tal\n","Isgd":"http:\/\/is.gd\/tgznfe","Lnkby":"http:\/\/LNK.by\/fg8dn5","Sinaweibo":"http:\/\/t.cn\/zOZrqTe","Zzb":"http:\/\/zzb.bz\/nL2eQ","Vgd":"http:\/\/v.gd\/sxEfh7","Google":"http:\/\/goo.gl\/dX3vE","CL":"www.weibo.com\/gdxkc"}

错误代码分析：
-1：网址不符合规范
-2：网址没有被传入
-100：其他错误
(错误时其他值均为NULL,所有错误值为负数!)
